Sample Analysis Script

# Compute CAGR between two values
def cagr(begin_value, end_value, periods):
    return (end_value / begin_value) ** (1.0 / periods) - 1

# Example
print(f"CAGR: {cagr(1000, 1200, 2):.2%}")

Key Financial Formulae

Net Present Value (NPV):

\(\mathrm{NPV} = \sum_{t=0}^{T} \dfrac{C_t}{(1 + r)^t}\)

Compound Annual Growth Rate (CAGR):

\(\mathrm{CAGR} = \left(\dfrac{V_{\text{end}}}{V_{\text{start}}}\right)^{\tfrac{1}{n}} - 1\)
